# Trainer Session Attendance Dashboard

This project is a React + Vite dashboard for tracking trainer session attendance.

## Features

- Transparent dashboard layout with a collapsible menu bar
- Attendance summary cards
- Batch-wise report panel
- Batch attendance chart
- Editable student attendance table
- Notification panel with sound toggle
- Profile card and chat box for quick session updates

## Run the Project

```bash
npm install
npm run dev
```

## Build for Submission

```bash
npm run build
```

## Notes

- The project has been verified with a successful production build.
- Main source files are in `src/App.jsx`, `src/App.css`, and `src/index.css`.
